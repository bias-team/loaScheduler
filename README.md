# loaScheduler
A toy project for scheduling and managing raid and participants using Flask and JSON, RESTAPI, etc...

# 프론트
## 기술 스택
- Vite
- React
- TypeScript
- Redux Toolkit
- React Router
- Axios

## 주요 변경사항
- Create React App에서 Vite로 개발 환경 마이그레이션
- TypeScript 적용
- 프로젝트 구조 변경:
  ```
  src/
  ├── components/
  ├── pages/
  ├── services/
  ├── store/
  ├── App.tsx
  └── main.tsx
  ```
